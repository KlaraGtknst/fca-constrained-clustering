\subsection{Banksearch Dataset}
\label{sec:banksearch_data}

We obtained the Banksearch dataset from Carnegie Mellon University.\footnote{\url{http://lib.stat.cmu.edu/datasets/} (29.12.2025)}
Each document is associated with exactly one topic, and the documents themselves are not hierarchically structured.
Bade et al.~\cite{bade_creating_2008} introduced a gold-label topic hierarchy to organize these topics.
Although the reported hierarchy contains 14 categories, only 11 topics are represented in the documents of the dataset.
The topic hierarchy, along with the number of documents per topic (given in brackets), is shown below.

    {\small

            \item Finance (0)
            \begin{itemize}
                \item Commercial Banks (1000)
                \item Building Societies (1000)
                \item Insurance Agencies (1000)
            \end{itemize}
            \item Science (0)
            \begin{itemize}
                \item Astronomy (1000)
                \item Biology (1000)
            \end{itemize}
            \item Programming (0)
            \begin{itemize}
                \item C/C++ (1000)

            \end{itemize}
            \item Sport (1000)
            \begin{itemize}

                \item Motor Racing (1000)
            \end{itemize}
        \end{itemize}
    \end{multicols}
    }
\end{center}
